Tell a Friend Tool

This tool has the basic layout for creating a tool. Once you have configured it for your specific needs you can iframe it into your website.

Configuring the tool.

Open the index.php file in your web editor like Dreamweaver and create the message you would like to send. You can also decide if you want users to be able to edit the message or leave it as is.

Open the config.php file and set the following parameters

email subject

That's it.
